# 🌐 VERTIX Website

Landing page oficial de VERTIX - Sistema de gestión empresarial para agencias y consultoras.

## 📋 Descripción

Sitio web corporativo de VERTIX con información sobre servicios, precios, casos de éxito y contacto.

## 🎯 Características

- 🎨 **Diseño Profesional** - UI moderna y elegante
- 📱 **Responsive** - Optimizado para todos los dispositivos
- ⚡ **Rápido** - Carga optimizada
- 🔍 **SEO** - Optimizado para buscadores
- 📊 **CRO** - Conversión optimizada

## 🛠️ Tecnologías

- **HTML5** - Estructura semántica
- **CSS3** - Estilos modernos
- **JavaScript** - Interactividad
- **Vercel** - Deploy y hosting

## 📦 Desarrollo Local

```bash
# Clonar
git clone https://github.com/nickmaukan/vertix-website.git
cd vertix-website

# Abrir en navegador
open index.html

# O usar servidor local
npx serve .
```

## 🚀 Deploy

### Vercel (Recomendado)

```bash
# Instalar Vercel CLI
npm i -g vercel

# Deploy
vercel
```

### GitHub Pages

```bash
# Push a branch gh-pages
git subtree push --prefix . origin gh-pages
```
