# VERTIX Templates

Plantillas de landing pages creadas con VERTIX.

## 📦 Plantillas Disponibles

### 1. Phone Store Colombia
**Nichos:** E-commerce de celulares, tecnología
**Características:**
- 8 productos con precios en COP
- Carrito funcional (localStorage)
- Filtros por categoría (Gama Alta, Media, Económica)
- Countdown timer para ofertas
- FAQ accordion
- Formulario de contacto + WhatsApp
- Responsive completo

**Ubicación:** `phone-store-colombia/`
